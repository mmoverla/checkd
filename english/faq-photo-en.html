FAQ - PHOTO



What is CHECKD® Photo?

CHECKD® Photo is a module in the CHECKD® Toolbox that makes it easy to register and to always be in control of deviations on a construction site.

You take a picture of the deviation with the application and send the deviation to the responsible contractor. The contracor have to approve the deviation and is necessary report when it will be fixed. When the deviation has been fixed, the contractor sends the deviation bck to the project manager.

The entire log is color coded so it will be easy to see how the deviation was handled.

How do you color code the log?

When the projectmanager sends the deviation it will be red.

When the resposible contractor recieves the deviation it will turn yellow.

In the end, when the deviation has been approved as done it will turn green.

How do I share a report with a team member?

In the application you need to find the report overview with a list of reports.

To the right of the report you want to share there is an icon you can push so that you send the report by e-mail.
